# Inventory_Management_Dashboard
Developed an interactive Inventory Management Dashboard in Power BI to monitor stock levels, inventory value, turnover ratio, and reorder alerts. The dashboard provides actionable insights for optimizing inventory control and improving operational efficiency.

# Tools Used:

1. Excel
2. Power BI

# Key Components of the Inventory Analysis Dashboard:
1. Total Stock
2. Total Products
3. Inventory Value
4. Total Quantity Sold
5. Estimation of Reorder Levels and Safety stock
6. Inventory Turnover Ratio
7. Stock Status
8. Safety Stock Levels

# Key Insights:
1. Top Selling Products and Low Selling Products
2. Monthly Trend
3. Inventory Value by Products
4. Top Revenue Products
5. Reorder alert table
6. Saftey stock alert table
7. High stock Low sales
8. Low Stock High Sales
   
# Visualization Used:
1. Bar chats
2. KPI Cards
3. Scatter plot
4. Table
5. Line chart
6. Tree map
7. Slicers
